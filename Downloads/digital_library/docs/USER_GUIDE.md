## Getting Started

### 1. Creating Your Account

**Step-by-step registration:**

1. Click on **"Register"** button in the navigation bar
2. Fill in your information:
   - **Full Name**: Your complete name
   - **Email**: Valid email address
   - **Password**: Strong password (minimum 6 characters)
   - **Role**: Select "Student" or "Educator"
   - **Institution**: Your school/university (optional)
3. Click **"Register"** button
4. You'll be automatically logged in

### 2. Logging In

1. Click on **"Login"** button
2. Enter your email and password
3. Click **"Login"**
4. You'll be redirected to the home page

### 3. Your Profile

- Click on your name in the top-right corner
- Select **"My Profile"** from the dropdown
- View and edit your profile information
- Update interests and preferences

## Finding Resources

### Basic Search

1. Use the search bar on the home page or any page
2. Type keywords (title, author, topic)
3. Press **Enter** or click **Search**
4. View results with resource details

### Advanced Search

**Using Search Page:**

1. Go to **"Search"** page from navigation
2. Use the filters on the left:
   - **Category**: Browse by subject area
   - **Level**: Choose difficulty (Beginner to Professional)
   - **Resource Type**: Select what you need
   - **Subject**: Specific subject area
   - **Language**: Filter by language
3. Combine filters for refined results
4. View results and sort by relevance

### Search Result Information

Each resource shows:
- **Title and Description**: What the resource is about
- **Author**: Who created it
- **Category**: Subject area

## Accessing Resources

### Viewing Resource Details

1. Click on any resource in search results
2. View the detailed page with:
   - Full description
   - Resource metadata

   - User reviews and ratings
   - Download and view statistics

## Reviews & Ratings

### Reading Reviews

1. Go to resource detail page
2. Scroll to **"Reviews"** tab
3. View community reviews and ratings
4. See ratings, titles, and comments
5. Check helpful/unhelpful votes

### Writing a Review

**Requirements**: Must be logged in

1. Go to resource detail page
2. Click on **"Write a Review"** tab
3. Fill in the form:
   - **Rating**: Select star rating (1-5)
   - **Review Title**: Brief title
   - **Comment**: Your detailed review (thoughts, pros, cons)
4. Click **"Submit Review"**
5. Your review appears in the reviews section

**Tips for good reviews:**
- Be honest and constructive
- Share specific examples
- Help others make decisions
- Keep it professional

## Uploading Resources (Educators)

### Upload Your Materials

**Educator Role Required**

1. Go to **"Upload"** option (if educator)
2. Fill in resource information:
   - **Title**: Clear, descriptive title
   - **Description**: Detailed description (100+ words helpful)
   - **Author**: Your name or organization
   - **Category**: Select primary category
   - **Resource Type**: Type of material
   - **Subject**: Subject area
   - **Level**: Difficulty level (Beginner to Professional)
   - **Tags**: Keywords for searchability
   - **Language**: Language of the resource
   - **Cover Image**: Optional cover image
3. Upload file (multiple formats supported)
4. Click **"Upload Resource"**
5. Resource enters approval queue
6. Admins review and approve
7. Once approved, becomes visible to users

### Resource Guidelines

**Best Practices:**
- Clear, descriptive titles
- Detailed, helpful descriptions
- Relevant tags and keywords
- Accurate metadata
- Quality materials
- Proper file formats

**Acceptable File Types:**
- PDF, DOC, DOCX
- PPT, PPTX
- XLS, XLSX
- Images (JPG, PNG)
- Videos (MP4, MOV)
- ZIP archives

`**File Size Limits:**`
- Maximum 100 MB per file
- Compress large files before uploading

- **Save searches**: Bookmark useful search URLs

### Organizing Resources

### Getting Quality Resources

- **Read reviews**: See what others say

- **Download samples**: Preview before full download

## Troubleshooting

### Can't Login?

- **Forgot Password?**: Use "Forgot Password" link
- **Check email**: Verify you're using correct email
- **Clear cookies**: Try clearing browser cookies
- **Try different browser**: Test in another browser

### Can't Find Resources?

- **Try broader search**: Use fewer filters
- **Check spelling**: Verify keywords
- **Use synonyms**: Try different keywords
- **Browse categories**: Browse by topic

### Download Issues?

- **Check file size**: Ensure disk space available
- **Try again later**: Try downloading after a few minutes
- **Use different browser**: Try Chrome/Firefox/Edge
- **Check internet**: Ensure stable connection

### Upload Problems?

- **File size**: Keep under 100 MB
- **File format**: Use supported formats
- **Complete information**: Fill all required fields
- **Wait for approval**: Admin review takes time
